HERO BANNER AND PAGE PATTERN

background-image background-size background-position


Use background-image to place a large photo in the header as a hero banner. The property background-size: cover scales the image to fill the entire element, cropping if necessary. The property background- position: center keeps the image centered as it crops. And background-repeat: no-repeat prevents the image from tiling.


The header needs a min-height so the background image has room to display.


For the page background, use a small tileable pattern image on the body. The property background-repeat: repeat tiles the pattern across the entire page. Adding background-attachment: fixed locks the pattern in place so it does not scroll with the content — the page content slides over the pattern.


Find a hero photo at unsplash.com Find a tileable pattern at subtlepatterns.com. Save both images to your project folder.
